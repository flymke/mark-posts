Mark Posts Plugin for WordPress

Mark Posts plugin provides an easy way to mark and highlight posts, pages and posts of custom post types within the posts overview.

Features

  • Set up maker categories
  • Assign marker categories to posts
  • view and edit them in your posts/pages or custom post types overview

Screenshots

1. Overview in WordPress Admin Panel

2. Set your Marker right on the edit post/page/custom post type screen

3. Set and configure ever Marker to fit your needs

4. Use the Quick Edit & Bulk Edit functions for quick and fast Marker updates

5. Check the status of your Markers right on the Dashboard

Installation

  1. Unzip the download package
  2. Upload mark-posts to the /wp-content/plugins/ directory
  3. Activate the plugin through the 'Plugins' menu in WordPress
